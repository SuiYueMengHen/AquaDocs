TechDocs 介绍

TechDocs 是一个现代化的技术文档平台，专为开发者和技术写作者设计。它提供了丰富的功能来创建、展示和分享技术文档。

主要特性

  • 完整的Markdown支持
  • LaTeX数学公式渲染
  • 代码语法高亮
  • 响应式设计
  • 明暗主题切换
  • 丰富的动画效果

使用教程

本章将介绍TechDocs的基本使用方法。

创建新文档

要创建新文档，请按照以下步骤操作：

// 示例：创建文档对象
const doc = new TechDocs.Document({
    title: "我的技术文档",
    author: "开发者",
    version: "1.0.0"
});

// 添加内容
doc.addSection("介绍", "这是文档的介绍部分。");
doc.addSection("安装", "安装指南内容...");

// 保存文档
doc.save();

示例代码

本章提供一些使用TechDocs API的示例代码。

Python 示例

import techdocs

# 创建文档
doc = techdocs.Document(title="Python示例文档")

# 添加内容
doc.add_heading("Python 示例", level=1)
doc.add_paragraph("这是一个使用TechDocs的Python示例。")

# 添加代码示例
code = """
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
"""

doc.add_code(code, language="python")

# 保存文档
doc.save("example.html")

JavaScript 示例

// 创建TechDocs实例
const techDocs = new TechDocs({
    theme: 'light',
    enableMath: true,
    codeHighlight: true
});

// 加载文档
techDocs.loadDocument('document.md')
    .then(document => {
        // 渲染文档
        techDocs.render(document, '#doc-container');
        
        // 添加事件监听
        techDocs.on('section-click', (sectionId) => {
            console.log(`Section clicked: ${sectionId}`);
        });
    })
    .catch(error => {
        console.error('Failed to load document:', error);
    });

帮助中心

常见问题和使用帮助

如何开始使用TechDocs？ +

首先，从下载页面获取适合您操作系统的版本。安装完成后，您可以查看文档页面的"快速开始"部分，或者直接创建新文档开始使用。

TechDocs支持哪些Markdown扩展？ +

TechDocs支持标准Markdown以及以下扩展：表格、任务列表、定义列表、脚注、缩写等。此外，我们还支持LaTeX数学公式和Mermaid图表。

如何切换明暗主题？ +

您可以通过以下两种方式切换主题：

  1. 点击页面右上角的主题切换按钮
  2. 在设置页面选择喜欢的主题
TechDocs支持导出哪些格式？ +

目前TechDocs支持导出为以下格式：HTML、PDF、Markdown和纯文本。我们计划在未来版本中添加对Word和Epub格式的支持。

如何报告问题或请求功能？ +

如果您遇到问题或有功能建议，可以通过以下方式联系我们：

  • 在GitHub仓库提交Issue
  • 发送邮件至 support@techdocs.dev
  • 加入我们的Discord社区
TechDocs是免费的吗？ +

是的，TechDocs是完全免费和开源的，遵循MIT许可证。您可以自由使用、修改和分发它。
